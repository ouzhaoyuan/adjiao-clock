# 阿刁时钟 ⏰

悬浮窗实时毫秒时钟 — 打开即显示，双击即关闭。

## 功能

- **实时时钟**：显示 `HH:mm:ss.SSS`（精确到毫秒）
- **零按钮**：打开APP直接启动悬浮窗，无需任何点击
- **双击关闭**：双击悬浮窗即可关闭
- **50%屏宽**：悬浮窗宽度为屏幕50%，居中显示
- **半透明背景**：70%透明度，不遮挡操作
- **等宽字体**：MONOSPACE字体 + 固定格式，防止数字闪烁
- **高刷新率**：~60fps更新，毫秒显示流畅

## 安装

1. 下载最新 APK：[Releases](https://github.com/ouzhaoyuan/adjiao-clock/releases)
2. 安装到手机（需允许「安装未知来源应用」）
3. 首次打开会请求「显示在其他应用上层」权限
4. 授权后重新打开即可

## 使用

- **显示时钟**：打开APP → 悬浮窗自动出现
- **关闭时钟**：双击悬浮窗 → 自动消失

## 权限说明

| 权限 | 用途 |
|------|------|
| SYSTEM_ALERT_WINDOW | 在其他应用上方显示悬浮时钟 |

仅此一项权限，无网络、无存储、无后台数据收集。

## 图标

应用图标使用 Python Pillow 程序自动生成（蓝色时钟造型），无第三方版权依赖。
